The command line arguments can be used to control the simulations so
as to gather data as plotted in the publication, for example to vary
calcium input and to vary PKC number. Command line arguments are also
used to specify what data to record and (optionally) to plot. The
usage of command line arguments is described further within the
simulation script.

Please refer to the STEPS website for more information about the
simulation environment and installation instructions:

http://steps.sourceforge.net

Python dependencies to run this script: Numpy, SciPy, Matplotlib.
