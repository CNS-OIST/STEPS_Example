How to use

Serial reaction-diffusion simulation using Tetxact: 
python ssa_simulation.py

Parallel reaction-diffusion simulation using TetOpSplit: 
mpirun -n 5 python parallel_simulation.py

Parallel reaction-diffusion-efield simulation using TetOpSplit:
mpirun -n 5 python HH_APprop_tetopsplit.py
